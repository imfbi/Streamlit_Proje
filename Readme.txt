
## Kurulum Talimatları

Projeyi .zip olarak indirin ve bir klasöre çıkartın.
Anaconda veya Python sanal ortamınızı kullanarak gerekli kütüphaneleri kurun.
Projeyi başlatmak için terminal veya komut istemcisinde "streamlit run main.py" komutunu kullanın.

## Kullanım

Uygulama başladığında, sol taraftaki yan menüden veri seti seçilir.
Breast Cancer Wisconsin Data ve Diabets olmak üzere iki veri seti bulunmaktadır.
Daha sonra sol taraftaki yan menüden bir makine öğrenimi modeli seçilir.
KNN, SVM ve Navie Bayes olmak üzere üç çeşit model bulunmaktadır.
Model seçildikten sonra, otomatik olarak en iyi parametrelerle eğitilir ve sonuçlar gösterilir.

## Kullanılan Araçlar ve Kütüphaneler
-pip install streamlit
-pip install pandas
-pip install seaborn
-pip install scikit-learn
-pip install matplotlib

- Streamlit: Web uygulamaları oluşturmak için kullanılan bir Python kütüphanesi.
- Pandas: Veri manipülasyonu ve analizi için kullanılan bir kütüphane.
- Seaborn: Veri görselleştirmesi için kullanılan bir Python kütüphanesi.
- scikit-learn: Makine öğrenimi modelleri oluşturmak ve değerlendirmek için kullanılan bir kütüphane.
- Matplotlib: Grafik ve görselleştirmeler oluşturmak için kullanılan bir kütüphane.
